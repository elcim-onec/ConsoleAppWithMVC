# Sentiment Analysis Console App using JDBC

## How it Works :point_down:	

#### 1. Enter your DB info

#### 2. Create data for 
- | 1 | Score
- | 2 | Sentiment
- | 3 | Comment Segment 
- | 4 | Product

#### 3. Perform Crud Operations by combining Table & CRUD number: (Ex 1-2 --> Score - GET BY ID)
- | 1 | GET ALL
- | 2 | GET BY ID
- | 3 | CREATE
- | 4 | UPDATE
- | 5 | DELETE
